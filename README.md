# SPLTECSOL — Official Corporate Website

A fully responsive, single-page corporate website built for **SPLTECSOL (Splendid Technologies)**, 
an enterprise IT infrastructure and physical security company based in Hyderabad, India.

## Tech Stack

- **HTML5** — Semantic, accessible markup
- **CSS3** — Custom properties, Grid, Flexbox, keyframe animations
- **Vanilla JavaScript** — Zero dependencies, lightweight and fast

## Features

- Responsive design across mobile, tablet, and desktop
- Animated client logo marquee with 10+ enterprise clients
- Service showcase with detailed capability breakdown per offering
- Quote request modal with dynamic service pre-selection
- Scroll-triggered reveal animations using Intersection Observer API
- CSS-only marquee loop animation
- Custom color palette derived from brand identity

## Sections

- Hero with stats
- Services with expandable detail cards
- Process / How We Work
- About
- Industries Served
- Clientele
- Call to Action
- Footer with contact details

## Deployment

Hosted via GitHub Pages with a custom domain configured through Zoho Domains DNS.
